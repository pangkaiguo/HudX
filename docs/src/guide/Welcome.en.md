# Welcome to HudX

HudX is a high-performance, lightweight charting library built on top of `hudx-render` and `hudx-charts`. It provides a comprehensive set of charts with interactive features, themes, and internationalization support.

## Key Features

- **High Performance**: Optimized for handling large datasets with efficient rendering (Canvas/SVG).
- **Interactive**: Built-in support for tooltips, legends, zooming, and animations.
- **Customizable**: Flexible theme system with light/dark mode and custom color palettes.
- **Internationalization**: Native support for multiple languages (English, Chinese, etc.).
- **React Ready**: dedicated `HChart` component for seamless React integration.
