# 欢迎使用 HudX

HudX 是一个基于 `hudx-render` 和 `hudx-charts` 构建的高性能、轻量级图表库。它提供了一套完整的交互式图表组件，支持丰富的主题定制和国际化。

## 核心特性

- **高性能**: 针对大数据集优化，支持 Canvas/SVG 双模式渲染。
- **交互丰富**: 内置提示框、图例、缩放和流畅动画支持。
- **高度定制**: 灵活的主题系统，支持亮色/暗色模式及自定义色板。
- **国际化**: 原生支持多语言（英文、中文等）。
- **React 友好**: 提供 `HChart` 组件，实现与 React 的无缝集成。
