## üìä Feature Overview

### Animation System
- Multiple easing functions (linear, quadratic, cubic, elastic)
- Delay and duration control
- Update and complete callbacks
- Pause/resume functionality

### Tooltip Component
- Show/hide functionality
- Custom styling
- Multi-line text support
- Position control

### Legend Component
- Horizontal/vertical layout
- Click to toggle visibility
- Selection callbacks
- Custom styling

### Event Handling
- Mouse events (mouseover, mouseout, click, dblclick)
- Touch events (touchstart, touchmove, touchend)
- Drag events (drag, dragend)
